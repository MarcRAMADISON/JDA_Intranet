<h2>JDA Intranet</h2>

<h3>Présentation </h3>

JDA Intranet est le CRM de JDA diffusion qui permet de gérer ses clients. Il présente plusieurs fonctionnalités :</br></br>
. Gestion des connexions ( login , signIn, change password )</br>
. la gestion de contacts (ajout, modifier,  supprimer) </br>
. Envoyer des documents à sign directement sur la plateforme ( yousign)
</br>. tableau de bord pour suivre la performance de chaque agent ou avoir un vie d'ensemble sur l'equipe</br>
. un dashboard pour gérer les comptes des agents en tant qu'administrateur 

<h2>Point technique </h2>
Technologies utilisées:</br>
Front-end : Nextjs / React material UI</br>
Backend : Strapi CMS v4
